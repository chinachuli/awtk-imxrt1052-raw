## input\_method\_t
### 概述
 输入法接口。

 常见的实现方式有以下几种：

 * 空实现。用于不需要输入法的嵌入式平台。

 * 缺省实现。用于需要输入法的嵌入式平台。

 * 基于SDL实现的平台原生输入法。用于桌面系统和手机系统。
