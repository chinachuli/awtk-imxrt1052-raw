## object\_default\_t
### 概述
![image](images/object_default_t_0.png)


 对象接口的缺省实现。

 内部使用有序数组保存所有属性，可以快速查找指定名称的属性。

#### object\_default\_find\_prop 函数
-----------------------

* 函数功能：

> <p id="object_default_t_object_default_find_prop">

* 函数原型：

* 参数说明：

| 参数 | 类型 | 说明 |

| 返回值 | value\_t* | 返回属性的值。 |
| obj | object\_t* | 对象。 |

| data | const void* | 要比较的数据。 |
