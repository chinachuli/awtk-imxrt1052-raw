## style\_mutable\_t
### 概述
![image](images/style_mutable_t_0.png)


 可变的style(可实时修改并生效，主要用于在designer中被编辑的控件，或者一些特殊控件)。

 style\_mutable也对style\_const进行了包装，当用户没修改某个值时，便从style\_const中获取。
