## theme\_t
### 概述
 主题。

 负责管理缺省的主题数据，方便实现style\_const。

| 函数名称 | 说明 |

| <a href="#theme_t_theme_create">theme\_create</a> | 创建主题对象。 |
| <a href="#theme_t_theme_deinit">theme\_deinit</a> | 析构主题对象。 |
| <a href="#theme_t_theme_destroy">theme\_destroy</a> | 析构并释放主题对象。 |
